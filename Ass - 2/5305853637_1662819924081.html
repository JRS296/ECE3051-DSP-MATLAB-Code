Use Matlab to solve thsi problem. use fft function.need matlab code.



Matlab Code:

n = 0:255;
f1 = 1/18; f2 = 5/128;
fc = 50/128;
xn = cos(2*pi*f1*n)+cos(2*pi*f2*n);%x[n]
xcn = cos(2*pi*fc*n);%xc[n]
xam = xn.*xcn;%xam[n]
figure(1);
subplot(3,1,1);
stem(n,xn);
subplot(3,1,2);
stem(n,xcn);
subplot(3,1,3);
stem(n,xam);
figure(2);
dftxam1 = fft(xam(1:128),128);%calculating 128 point dft for 0<=n<=127
plot(abs(dftxam1));
figure(3);
dftxam2 = fft(xam(1:100),128);%calculating 128 point dft for 0<=n<=99
plot(abs(dftxam2));
figure(4);
dftxam3 = fft(xam(1:180),256);%calculating 256 point dft for 0<=n<=179
plot(abs(dftxam3));
